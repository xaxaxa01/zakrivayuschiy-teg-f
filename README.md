# Закрывающий тег

Адаптивная учебная страница с CSS-анимациями.

## 📌 О проекте
Проект сверстан по учебному макету.  
Реализована семантическая структура текста, блоков и изображений.  
Проект адаптирован под разные устройства.  
Добавлены CSS-анимации для интерактивных элементов и плавного отображения контента.

## 🛠 Использованные технологии
- HTML5
- CSS3
- Flexbox
- БЭМ-методология
- CSS-анимации и hover-эффекты

## 📱 Реализовано
- Адаптивная верстка 
- Семантическая разметка текста и блоков  
- Работа с изображениями  
- CSS-анимации и hover-эффекты  
- Чистая и структурированная верстка
